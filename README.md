# Flashlight App

A simple Android application built as a self-learning project to understand Android development fundamentals using Kotlin and XML. The app allows users to toggle the device flashlight, records the timestamp of each ON/OFF event, and stores the usage history locally using Shared Preferences.

> **Project Status:** This project is under active development and is continuously being updated with new features, improvements, and Android best practices as I continue learning.

## Tech Stack

* **Language:** Kotlin
* **UI:** XML, ConstraintLayout, Material Design Components
* **Platform:** Android SDK
* **Storage:** Shared Preferences
* **API:** CameraManager API
* **IDE:** Android Studio
* **Build Tool:** Gradle

## What I Learned

* Android project structure and application lifecycle
* Building user interfaces using XML and ConstraintLayout
* Working with Activities and Intents
* Requesting and handling Android runtime permissions
* Accessing device hardware using the CameraManager API
* Saving and retrieving persistent data with Shared Preferences
* Recording and displaying usage history
* Implementing Material Design components
* Testing and debugging Android applications on a real device
* Using Git and GitHub for version control and project management
